# Frontend

This folder contains the WorkVerse client application built with React and Vite.

## Responsibilities

- Public landing page and jobs listing
- Login and registration flows
- Candidate profile experience
- Employer dashboard and candidate view
- Admin dashboard, query console, and graph explorer

## Main Entry Points

- `src/App.jsx` defines the application routes
- `src/api.js` configures API communication with the backend
- `src/pages/` contains route-level screens
- `src/components/` contains shared UI pieces

## Run Locally

```bash
npm install
npm run dev
```

The app expects the backend API at `http://localhost:5000/api`.
